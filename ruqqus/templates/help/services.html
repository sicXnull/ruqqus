These services may see your personal information. This list is subject to change without notice, but will always be kept up to date. These services are not affiliated with Ruqqus LLC. {% endfilter %}
Service Function About
Heroku Origin Server Heroku hosts the core Ruqqus server, which processes web requests. Heroku itself runs on AWS.
Heroku Postgres SQL Database The primary data store for all Ruqqus user and content information.
Amazon S3 Image Hosting Amazon's S3 storage service acts as the host for i.ruqqus.com, where all user-submitted images are stored.
Cloudflare DNS service and DDOS prevention Cloudflare acts as an edge network, caching certain types of content closer to the end user, and preventing DDOS attacks.
Mailgun Email processing Mailgun provides an API service for automated sending of email. Ruqqus uses this for email verification and account recovery.
LogEntries Extended Logging LogEntries provides storage of server logs up to 7 days. This helps us track down some of the less easily noticable bugs.
New Relic Optimization analytics New Relic provides server-side analytic tools to help identify and improve inefficiencies in the Ruqqus systems.
